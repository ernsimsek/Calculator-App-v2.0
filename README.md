# Calculator 2.0

A modern, multi-functional web calculator built with **HTML, CSS, and Vanilla JavaScript**.  
Calculator 2.0 is designed as an all-in-one tool that combines scientific calculations, graph plotting, programmer utilities, unit conversion, voice input, themes, and offline support in a clean glass-style UI.

---

##  Features

###  Standard & Scientific Calculator
- Basic operations: addition, subtraction, multiplication, division, modulo
- Scientific functions:  
  `sin`, `cos`, `tan`, `log`, `ln`, `√`, `x^y`, `π`, `e`, factorial
- Degree / Radian mode toggle
- Keyboard support
- Calculation history with export & clear options

###  Graphing Calculator
- Plot mathematical functions such as:
  - `sin(x)`
  - `x^2`
  - `log(x)`
- Zoom in / Zoom out / Reset view
- Dynamic canvas resizing

###  Programmer Mode
- Base conversions: **HEX / DEC / OCT / BIN**
- Bitwise operations:
  - AND, OR, XOR, NOT
  - Left Shift, Right Shift
- Real-time number base display

###  Unit Converter
Supports multiple categories:
- Length
- Weight
- Temperature
- Data Size
- Currency (live exchange rates with caching)

###  Voice Input
- Speak simple math expressions in English
- Example: “five plus six equals”

###  Theme System
- Dark
- Light
- Neon (Cyberpunk)
- Pastel  
Themes are saved in local storage.

###  History Panel
- Stores last 50 calculations
- Click to reuse results
- Export history to `.txt` file
- Clear history option
- Notification indicator for new entries
